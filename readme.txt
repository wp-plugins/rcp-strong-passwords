Enforce strong passwords during user registration for Restrict Content Pro

== Description ==

This is a simple extension for [Restrict Content Pro](http://pippinsplugins.com/restrict-content-pro-premium-content-plugin/) to enforce strong passwords during registration. This extension does NOT function without Restrict Content Pro installed.

This extension has no options and will automatically require users to use a strong password during registration.

Learn more about Restrict Content Pro: http://pippinsplugins.com/restrict-content-pro-premium-content-plugin/
